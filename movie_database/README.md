# 🎬 Movie Database using MongoDB

## 📌 Description
This project is a simple Movie Database built using MongoDB.  
It demonstrates basic database operations like Create, Insert, Read, Update, Delete, and Validation.

---

## ⚙️ Features
- Create collection with validation
- Insert data using insertOne and insertMany
- Read data using find and findOne
- Update data using updateOne and updateMany
- Delete data using deleteOne and deleteMany
- Validate and verify data

---

## 🛠️ Technologies Used
- MongoDB

---

## 📂 Project Structure
movie-database/
├── queries.js
└── README.md

---

## ▶️ How to Run
1. Open MongoDB shell or MongoDB Compass
2. Run the queries from queries.js
3. View results in the database
